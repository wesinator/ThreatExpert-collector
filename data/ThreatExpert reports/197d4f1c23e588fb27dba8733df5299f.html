Possible Security Risk

Security RiskDescription
Trojan-PWS.Sagi Trojan.PWS.Sagi is a Trojan which on execution installs itself as a legitimate program, runs in the background and monitors Yahoo! Messenger. It captures passwords entered in the Yahoo! Messenger windows and sends those passwords to the author of this Trojan.
Application.WinSession_Keylogger WinSession Keylogger is a Monitoring software from Cromosoft Technologies. It can run in stealth mode and capture all keystrokes, applications used, emails, text and images from clipboard, chat sessions, instant messages, websites visited and can take screenshots of the system at regular intervals or mouse clicks. It has content filters which can be configured to block websites and applications. All the information captured is stored by this software in log files and it has the ability to send these log files to a specified email address. Removal of this software is advisable if it is not installed for a purpose.
