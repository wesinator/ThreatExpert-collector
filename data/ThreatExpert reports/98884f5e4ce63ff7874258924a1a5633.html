Possible Security Risk
  • Attention! Characteristics of the following security risks were identified in the system:

Security RiskDescription
Worm.Mydoom Worm.Mydoom is a mass-mailing worm. It uses the infected machine to send outgoing emails and allows unauthorized access to the infected machine.
Worm.Mydoom.BG This worm drops a copy of itself in the WINDOWS SYSTEM directory as KERNEL_RUNTIME.EXE. It also drops a text file KERNEL_RUNTIME.EXE-UP.TXT. Worm.Mydoom.BG propagates by attaching a copy of itself in an email. Worm.Mydoom.BG is also capable of connecting to the Internet Relay Chat (IRC) server where it listens for remote backdoor commands.
